1. AJAX - асинхронный js и xml, асинхронно отправляет http-запросы серверу и обрабатывает его ответы
2. WebSocket - технология, устанавливающая соединение между клиентом и сервером через TCP
Это позволяет им взаимодействовать в реальном времени без обновлений страницы и постоянных запросов 
3. WebSocket API - программный интерфейс, основанный на полудуплексной связи WebSocket
Необходим для обмена сообщениями между клиентом(браузером) и сервером
4. Функции WebSocket API
- создание вебсокет-соединения
- отправка и получение данных
- обработка событий
- управление соединением
- обработка ошибок
5. Дополнительные возможности WebSocket API
- использование протоколов для передачи данных
- защита соединений(ssl/tsl)
- отправка данных в виде фрагментов
